# PATINHO-BOARD

## overview
patinho board é uma app de perguntas e respostas de código feita em parceria com inteligencia artificial para colaboração mutua entre pessoas desenvolvedoras de código e lógica de programação 

## técnologias, frameworks e tools 
- javascript
- node js
- vainilla web
- sql usando sqlite com knex 
- mongo usando mongoose
- ai utilizando copilot github agent como debugger e improvement de código e ideias
